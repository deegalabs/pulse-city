brand

A living soundtrack for a place — a 24/7 AI-composed, code-visible radio station for Ipê Village 2026. The editor is one surface; the radio is another. Both expose the same code.

purpose

To give a place a soundtrack that's awake — and to keep it playing.

vision

Every pop-up city, co-living space, residency, and long-form event has its own pulse.city station. The world is quieter, but more attentive.

mission

Build a 24/7 AI-composed, code-visible radio station for Ipê Village 2026 that residents can tune and co-create without touching a DAW.

values

value 01
the code is the voice

Every marketing visual includes real Strudel code. The hero is four lines that actually play.

value 02
awake, not hype

One dot pulsing at 1.5s. A spectrum breathing at real audio rate. A manifesto line, not a paragraph.

value 03
the place remembers

Listener state persists. Returning visitors see "welcome back" by state, not modal. History is browsable.

value 04
ration the loud

/studio is loud. /radio is quiet. Same brand, different volume. Uniformity applied without taste is a tell.

manifesto

the city is playing. the code is the voice. the place is awake, and it remembers you were here. you can edit any line. the station keeps going.

voice

Present · Legible · Understated. Confidence without volume. The product is loud; the voice is not.

three voice words

voice 01
present

Current tense. "The city is playing." Not "will play." Not "could play." It is happening.

voice 02
legible

No jargon unless the jargon is the point. Code is OK; "synergy" is not. Sentences are short.

voice 03
understated

Confidence without volume. No exclamation marks. No all-caps hype in prose. Chrome labels only.

seven voice rules

  1. Never use ! in marketing copy. Chrome labels ([ERR], PLAYING) can use all-caps — they are functional.
  2. Never use the forbidden words: "vibes," "drop," "banger," "epic," "game-changer," "revolutionary," "next-gen," "paradigm shift."
  3. Prefer verbs over nouns. "The station plays" > "station playback engaged."
  4. Prefer concrete over abstract. "The code is the voice" > "Our innovative approach."
  5. Acknowledge the listener as a person, not a user. "You were here" > "The user is here."
  6. Code blocks are part of the copy. A marketing page can show 4 lines of real Strudel and that IS the pitch.
  7. Ration the loud. Uppercase tracked-widest is a powerful tool — use it on /studio, barely on /radio, never on /p.

whitespace claims

  • The only transparent station. Every competitor is either a black-box stream or a visible-code track.
  • The studio is the radio, paused. One surface, two modes. You never "publish." You stop, and the city keeps playing what you left it.
  • The code is the voice. No cover art, no mood screenshots, no marketing visuals that are not also functional code.

logo

A 3×7 punchcard. Twenty-one dots, one saturated. The center dot is listener sky — the only bright dot in the grid. The mark says awake, and remembers you were here: still, but with one bright acknowledgement.

rules

  • Clear space: 1u = 16px (one dot cell) on all sides.
  • Minimum: full lockup 112×20 at display; mark-24 at favicon; ASCII in logs.
  • Never rotate, recolor (except the inverted variant), animate the outer dots, or embed in gradients.
  • The center dot is the only saturated dot. Never duplicate saturation elsewhere in the grid.
  • Prose form is pulse.city (lowercase, dot). Display form is PULSE·CITY (middot).

hard rules

  • No gradients anywhere — not hero, not buttons, not chrome.
  • No pastel tints. No creator-100, no listener-50.
  • No second border color. One value, period.
  • Page background floor: #0a0e17. Nothing darker. Surface ramps go up.
  • Max one signal-warn firing per screen. Amber is precious.
  • Agent never renders as long-form body text on surface-1 (3.6:1). Chrome only.

visual elements

Two motion shapes. One static dither. One icon library. No stock photos. No aesthetic code backdrops. The imagery comes from @strudel/draw output — the music visualizes itself.

dither texture

base · 2%
surface-1 · 2%
surface-2 · 4%
surface-3 · 0%

Applied as a 4×4 SVG tile, image-rendering: pixelated. Static, never animated. Zero on surface-3 — the /radio quiet tier.

motion vocabulary (exactly two)

pulse-dot · 1.5s · infinite · ambient
s("bd*4") evolve-glow · 600ms · one-shot · mutation

Nothing else moves. No hover bounces. No scroll parallax. No micro-interactions on every button. The brand is still, except when something happened.

iconography — Lucide

Library: Lucide (ISC, AGPL-compatible). Stroke 1.5px on 24px canvas — slightly thinner than default 2px, matching the understated voice. Outline only, no filled variants. Color inherits from currentColor.

imagery sources

  • Real @strudel/draw output: piano rolls, oscilloscopes, pattern visualizations captured from the running station.
  • Subtle 1-bit dither on dark surfaces — static, 2–4% opacity.
  • OG images: base + 4% dither + real piano roll + wordmark + one manifesto line.
  • Stock photos, AI-generated faces, illustration.
  • "Aesthetic code" backdrops, blurred code screenshots, code as decoration.
  • Auto-playing hero video, particle backgrounds, gradient blobs.

personas

Three people this brand has to earn. Lia — the co-creator resident. Oscar — the live-coder skeptic. The drifter — the silent majority who leaves /radio open in a tab.

primary · 60%
Lia

the co-creator resident

28, lives in Ipê Village for 6 weeks. Has a laptop, not a DAW. Curious about live coding but will not learn Tidal notation. Opens /studio on her phone to tweak the mood.

what she needs: the code to feel legible without a manual.

secondary · 30%
Oscar

the Strudel live-coder

34, five years of Tidal. Skeptical of AI-composed music. Will fork the repo. Wants to run a set at the Village café and have the radio hand him the mic.

what he needs: a credible invitation — not "we replaced you," but "we kept it playing until you arrived."

tertiary · the silent majority
The drifter listener

the silent majority

Anyone who opens /radio in a background tab while they work. Knows nothing about code. Stays because the station has a pulse that matches the room.

what they need: a visual language that feels calm and alive at the same time — not a hype VJ set, not an ambient meditation app.

applications

Same brand, different volume. The hard rule: ration the loud. /studio shouts in functional chrome. /radio barely whispers. /p is editorial. /embed is a guest. marketing balances. MCP is documentation.
