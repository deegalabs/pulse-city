# Brand Platform
> Phase: strategy | Brand: pulse-city-next | Generated: 2026-04-15

---

## Purpose
So that any place — a kitchen, a coworking floor, a pop-up city — can have a soundtrack as open, readable, and alive as the people in it.

## Vision
A world where the default shared-soundtrack layer for intentional communities is an open primitive, not a subscription, and where every pattern on it carries the name of the hand (human or agent) that wrote it.

## Mission
Ship and maintain one living soundtrack primitive with four legible surfaces — /studio, /radio, /p, /embed — that humans and MCP agents compose on together, starting with Ipê Village 2026.

## Values

Four values. Each value has a "looks like / doesn't look like" pair so it can't be mistaken for a competitor's value.

### 1. Visible over magical
- **Looks like:** the agent's diff appears in the same editor Lia types in; the pattern code is on the /p page, above the fold.
- **Doesn't look like:** "AI-powered," a dedicated AI panel, a "generate" button with a sparkle icon.

### 2. Shared over personalized
- **Looks like:** /radio/ipe is the same song at the same second for everyone in the room; wall-clock sync.
- **Doesn't look like:** "made for you," "your daily mix," a per-listener recommendation engine.

### 3. Alive over finished
- **Looks like:** every pattern has "open in /studio" as a first-class affordance; leaving a loop running is a valid authorship move.
- **Doesn't look like:** "release a track," "album art," a version number on a pattern, a launch countdown.

### 4. Credited over owned
- **Looks like:** `pc_` MCP keys per author; agent commits show up with the agent's name; AGPL-3.0-or-later; forking is encouraged in the empty state copy.
- **Doesn't look like:** "pulse.city pro," a DMCA page, a proprietary stems format, a "featured by pulse.city" badge.

## Promise

- **Functional:** a living soundtrack primitive that runs itself, that anyone (human or agent) can shape, and that stays readable on every surface it lives on.
- **Emotional:** the place is awake, and it remembers you were here — contribute if you want, just listen if you don't, the pulse doesn't stop either way.

## Manifesto

> the city is playing.
> the code is on the screen, because the code is the song.
> any hand can pick up the loop — yours, a stranger's, an agent's.
> leave it running. step out. come back.
> the place is awake, and it remembers you were here.

Five lines. Present tense, lowercase, no exclamations, no first-person-plural advocacy. Every line is a description of what the product actually does — which is the whole point. Any of the five could sit under a wordmark on a poster.

## Brand heartbeat

**the place is awake, and it remembers you were here.**

This is the compass. Every other artifact in this strategy — the archetype, the voice, the messaging pillars, the naming rules — exists to make this sentence true on four surfaces at once. If a design decision cannot be traced back to "does this make the place feel more awake, or more remembering," it is not a pulse.city decision.
