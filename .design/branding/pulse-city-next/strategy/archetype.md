# Archetype
> Phase: strategy | Brand: pulse-city-next | Generated: 2026-04-15

---

pulse.city is **Creator, with Sage as the tension archetype.** Creator drives what the brand wants to be; Sage is the conscience that keeps Creator from drifting into self-expression-for-its-own-sake.

## Primary: Creator

Creator here is not the "unleash your inner artist" Pinterest version. It is specifically the live-coder's Creator: *if you can read it, you can make it — and if you can make it, you can leave it running for someone else to pick up*. Authorship is a gift, not a performance.

**Three Creator traits translated into product behavior:**

1. **Every pattern is forkable.** The Creator archetype here insists that nothing on screen is "finished." /p/[id] ships with an "open in /studio" button on principle — a listener is a Creator who hasn't started yet. Contrast: Suno's Creator is the one who typed the prompt and walked away. pulse.city's Creator is the one who walks in.
2. **The tool makes its own imagery.** Punchcard and scope visuals come from the pattern itself (Strudel `draw`). Creator doesn't decorate the work; the work decorates itself. This is why the brand has no stock photography rules — there is no stock photography.
3. **Leaving it running counts as authorship.** The Ipê Village success metric in the brief — "Lia leaves it running on a projector for an evening" — is a Creator move: the loop is part of the work. Radio is not consumption. Radio is slow authorship.

## Tension: Sage

Creator alone becomes precious — *look at this thing I made, please admire it*. Sage stops that. Sage in pulse.city means: **the truth is in the code, and the code is on the screen.** Knowledge is visible, revisable, and cheap to inspect. If a claim is not auditable, it is not a claim this brand makes.

Sage is why "AI-powered" is a forbidden phrase. Sage does not invoke magic — Sage shows the work. When an agent composes a pattern in autopilot, Sage says: here is the diff, here is the MCP key that authored it, here is the model, here is the prompt if you want it. Nothing hides.

**Why this specific pairing differentiates:**

- **Suno is Magician.** Closed transformation, surprise reveal, "how did it do that." pulse.city is structurally incapable of this move — Sage will not let it.
- **Endel is Caregiver.** Wellness posture, neuroscience-backed, take care of you. pulse.city does not take care of anyone — it plays, and you are welcome.
- **Strudel is pure Sage.** Research-shaped, lineage-shaped, documentation-heavy. pulse.city inherits the Sage rigor but Creator adds warmth, a venue, and a listener.
- **IPÊ.FM (sibling) is closer to Everyman + Sage** — the radio-for-the-room posture. Creator is what pulse.city adds on top so that a listener can become an author without leaving the URL.

## Archetype applied to product behavior

| Surface | Creator expression | Sage expression |
|---|---|---|
| **/studio** | Lia writes a pattern, hears it immediately, forks someone else's work in two clicks. The UI reads like a terminal that invites typing. | Code is the interface. No hidden state. The agent's diffs appear in the same editor the human types in. |
| **/radio/[name]** | The station is a long composition. Whoever curated it is credited. Slow Creator — authorship measured in hours, not beats. | Wall-clock sync: the same song at the same second for everyone in the room. No personalization, no black-boxed shuffle. |
| **/p/[id]** | Listener sees "open in /studio" as a first-class option. Sharing a pattern is offering an invitation to make it. | The pattern code is visible on the page. You can read it while you listen. There is no "view source" hidden in a menu — source is the page. |
| **/embed/[id]** | A host site gets a floating widget that carries its author's credit with it. The embed is a portable Creator signature. | The widget exposes which pattern, which author, which license. No tracking, no phone-home, no surprise network calls. |

## Archetype applied to MCP agents

This is the single biggest strategic payoff of picking Sage as the tension archetype.

**Agents are collaborators, not tools.** An agent shows up in /studio the way Lia shows up: with a key, a name, and a diff. The brand never says "AI-powered" and never uses "powered by." The agent is an author with a `pc_` key prefix, a model name, and a visible commit in the pattern history. Sage says: if we let a model write code, we credit it and we show the diff. Creator says: the model is a collaborator, which means it can be disagreed with, reverted, or remixed, same as any human Creator.

Concretely:
- The autopilot toggle is not "generate for me." It is "hand the loop to the agent." The handoff is visible.
- Every agent-authored diff shows up in the same editor Lia types in. No dedicated "AI" panel. (Dedicated panels are a Magician move.)
- The chat is not a prompt box. It is a conversation with a co-author who happens to be a model.
- The violet color (agent / autopilot / MCP) is warmed — closer to the compass — specifically so the agent does not read as machine-other. It reads as a second hand in the room.

## What Creator x Sage forbids

Bright-line no-gos. If a design move implies any of these, it is not pulse.city.

1. **No "AI genius" framing.** No "our model." No "trained on millions." No surprise reveal. Sage will not allow mystification.
2. **No gamification.** No streaks, no XP, no "listener level 3." Creator is not a loyalty program. The place is awake; it is not a game.
3. **No hidden state.** If it affects the sound, it is visible in the code. No EQ preset panel, no undocumented default tempo, no invisible "enhancement."
4. **No wellness-adjacency.** No "focus mode," no "sleep mode," no "science-backed." Endel owns that air and we do not want it. The place is a kitchen, not a clinic.
5. **No proprietary lock-in.** AGPL-3.0-or-later, no walled formats, no "pulse.city pro." If someone forks the whole thing, the compass says: good. That's the point.

## Shadows to watch

Every archetype has a shadow. We are going to hit both.

**Sage's shadow is lecturing.** Pulling a user into a four-paragraph explanation of what a cycle is, writing "well, actually" microcopy, turning the empty state into a tutorial. Catch it when: documentation leaks into UI, when microcopy explains instead of describes, when an error message tries to teach.

**Creator's shadow is preciousness.** "Every pattern is a work of art." Treating a pattern title as a poem instead of a filename. Writing marketing copy that gets emotional about the craft. Catch it when: a string wants to be admired, when a page starts with "we believe," when the voice slips into first-person plural to claim feelings.

The rule: **describe, don't decorate.** If a line reads like it is decorating the work, cut it. Sage says the truth is in the code; Creator says the work speaks for itself. Neither archetype needs the brand to do the speaking.
